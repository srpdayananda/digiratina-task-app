## How to run Backend(Express app)

        > go to backend folder from the terminal
        > npm install
        > npm run build
        > npm run dev


## How to run Frontend(Angular app)

        > go to frontend folder from the terminal
        > npm install
        > ng serve --open
        > angular app will be opened on browser
